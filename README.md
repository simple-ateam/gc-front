- 전국 야영장 지도 서비스
  > 공공 데이터 api과 naver map api를 활용하여 전국에 있는 야영장을 지도에 마커로 표시하고 그 정보를 제공해주는 서비스를 구현했습니다.

## Stacks

- React.js, Recoil, Emotion

## Points

- react v18, Suspense를 활용하여 비동기 컴포넌트 렌더링(스켈레톤 UI, 로딩 컴포넌트 등)을 구현.
- React-device-detect를 통해 접속 기기(브라우저/모바일)에 따라 다른 동작을 하도록 구현.
- UI Framework없이 모든 CSS를 직접 구현. emotion을 통해 주요 색상, 박스 사이즈, 폰트 사이즈 등을 theme화하여 일관된 스타일로 보여질 수 있도록 시도.
